Man Calc (manual calculator) is a calculator for iOS that can deal with very large or very small numbers (around 1000 decimal places of precision), and supports both infix and postfix syntax. Here are some examples. Type the stuff on the left. The symbol ↩ means "press enter" and the symbol ← means press delete". The following line shows the resulting stack between the square braces. The rightmost value in the stack is the result of the most recent operation.

In addition to the above, [Man Calc]> supports parentheses for adjusting order of operations, hexadecimal (0xbabecafe), octal (0o777), and binary (0b1010101) literals. Hex numbers can begin with '0x', 'x', '0h', or 'h'. Octal numbers can begin with '0o' or 'o'. Binary numbers can begin with '0b' or 'b.

The 'hex', 'oct', and 'bin' display modes only display the integral part of the number in the chosen base. The full precision is kept internally. Return to normal display of the full precision with 'dec' or 'eng'. If the number is too large to display in the chosen base, it will display as 'dec' would display it. This generally happens when 'dec' already displays the number in engineering notation.

Long running operations which display the animation can generally be stopped by pressing delete.
